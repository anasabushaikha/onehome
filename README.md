# OneHome Listing Filter

Live at **https://anasabushaikha.github.io/onehome/**

Paste your agent's OneHome share link to filter listings by condo type, owner-pays-water, parking, and in-suite laundry. See the [main project README](https://github.com/anasabushaikha/house-search) for how the filters are derived — no backend needed, this calls OneHome's API directly from the browser.
